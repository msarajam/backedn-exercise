# backend-exercise

## Required software

- make
- Go v1.12.*
    - https://golang.org/
- Dep v0.5.3
    - https://github.com/golang/dep/releases
- GolangCI-Lint
    - https://github.com/golangci/golangci-lint
- GoMock and `mockgen`
    - https://github.com/golang/mock        


## Testing

```bash
make test
```

## Running

```bash
make run
```

## Bundling

```bash
make bundle
```
